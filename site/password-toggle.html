PasswordToggle.js

Use PasswordToggle.js to append a show/hide toggle button to a text or password input element.

How it works

When a standard HTML <input> element of type text (initially visible) or password (initially hidden) is decorated as a PasswordToggle, the component wraps the input in an input-group element and appends a toggle button.

By default, the toggle button contains an icon (<i>) element styled with the classes fa-solid fa-eye when the password is hidden and fa-solid fa-eye-slash when the password is visible. These class names can be customized using the showIcon and hideIcon options or the data-show-icon and data-hide-icon data attributes.

Similarly, the toggle button itself can be customized using the toggleBtn option or the data-toggle-btn data attribute.

In an Input Group

When initialized, PasswordToggle creates an input-group wrapper around the target input and appends a toggle button. If the target input is already inside an input-group, PasswordToggle reuses the existing group rather than creating a nested input-group. This allows PasswordToggle to be combined with other input-group buttons and add-ons without disrupting the layout.

Options

Name Type Default Description
showIcon string 'fa-solid fa-eye' The CSS class names value to apply to the icon contained in the toggle button when the input content is hidden.
hideIcon string 'fa-solid fa-eye-slash' The CSS class names value to apply to the icon contained in the toggle button when the input content is shown.
toggleBtn string 'btn btn-outline-secondary' The CSS class names value to apply to the toggle button.

Methods

Method Description
toggle Toggles show/hide state.
destroy Removes the toggle button and restores the input element to its original location in the DOM.
