Countdown.js

This JavaScript module addes a character countdown under an input or textarea so end users know how many characters they can type.

How it works

This simple module creates a counter under a textarea or input that is updated everytime the element input event is fired.

Examples

Start by adding the initialization JavaScript to your page.

Simplest Setup

The max length defaults to 255 characters, the default length of a varchar column in the database.

Set Max Length

If you want to change that length, just set the maxlength attribute.

Change Label

You can change the label from 'Characters remaining: ' to whatever you want. Please remember to add a space at the end.

Auto Resize Height While Typing

This setting forces a textarea to resize based on the lines of text in it. The minium height is the rendered height. So if you set the rows or CSS height, that will be the minimum height.

Options

Unlike the other options where you convert the name to a data attribute (e.g. labelClass -> data-label-class), maxLength is just the attribute. This allows it to function like it always does in browsers.
Name Type Default Description
maxLength number 255 The maximum length the input (and by extension, the database column) can hold.
label string 'Characters remaining: ' The label of the counter.
labelClass string 'form-text' The CSS class on the div under the textarea/input.
showAll boolean false Toggles the auto resizing of the textarea when an end user types.
